# EasyDataTableEditor

The DataTable editor supports multi-line modification by right-clicking the file > Open EasyDataTableEditor

支持多选和多行修改的DataTable编辑器，打开方式：右键DataTable->Open EasyDataTableEditor,
支持DataTable和CompositeDataTable
